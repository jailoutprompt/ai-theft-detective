# AI 도난탐정 MVP 데모
작성: 2026-04-13

## 목표
창업중심대학 발표에서 라이브 구동 가능한 AI 도난탐정 웹앱 MVP

## 접근 방식
- Frontend: 단일 HTML (기존 앱 목업 디자인 활용, 다크테마)
- Backend: Python FastAPI (크롤링 + AI 유사도 분석)
- 크롤링: 당근마켓/번개장터 웹 검색 (Playwright)
- AI: OpenAI embeddings 기반 텍스트+이미지 유사도
- 지도: Kakao Map API (CCTV 위치 표시)

## 단계
1. [ ] FastAPI 백엔드 세팅 + 도난 신고 API
2. [ ] 중고마켓 크롤링 엔진 (당근마켓 검색)
3. [ ] AI 유사도 분석 (텍스트 매칭 + 점수)
4. [ ] 프론트엔드 대시보드 (도난신고 → 스캔 → 결과)
5. [ ] 112 신고서 자동 생성
6. [ ] CCTV 지도 (공공데이터 기반)
7. [ ] 이동 패턴 예측 (기존 .pkl 모델 연동 or mock)
8. [ ] 통합 테스트 + 데모 시나리오 검증

## 완료 조건
- 발표장에서 노트북으로 `localhost:8000` 접속
- "도난 신고" → 자전거 정보 입력 → "스캔 시작"
- 3~5초 후 "유사 매물 3건 발견!" 결과 표시
- 112 신고서 PDF 생성 + CCTV 지도 표시
- 전체 플로우 2분 이내 데모 가능
